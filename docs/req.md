完成一个科技感的waitlist页面，色调以白色和淡蓝色为主，UI风格模仿Apple的极简风格；
语言使用英文，不考虑国际化；
要分为Home、Blogs、ContractUs三大板块；

# Home 
Home页面参考desc.md中的内容，顶部是巨大的View区域和右侧waitlist的订阅栏目；
下方逐个模块介绍价值、主要用户以及其他关键亮点，也可以参考desc.md中的内容；
在home页面往下滑动到每个介绍部分的时候，加一些好看的过度效果；

# Blogs
Blogs模块展示一些发布日志，这些日志都是纯静态页面，目的是为SEO优化做准备；
Blogs列表中的内容和指向的路径在一个json文件中统一配置，在Blogs模块根据json文件加载清单；
当用户点击清单中的某一项时，跳转到对应的界面；
