# CUSTRA Waitlist 整体文案（含 SEO 关键词）📣

> 产品一句话：**CUSTRA 是一款本地运行的“会议 / 课程画面记录助手”**——你只需框选 Zoom/腾讯会议/Teams/B 站直播窗口中的一块区域（PPT、共享屏幕、字幕、聊天等），它就会按 **1 秒/次**对比“上一次保存帧”，在画面变化时自动保存关键截图；也支持“循环记录”定时兜底；可选录制系统输出音频（Windows）。

## 2) CUSTRA 解决的核心问题 🎯

### 会议/网课里最痛的不是“没内容”，是“来不及记”

- **你在听**：讲师/同事一页页翻 PPT、切 Demo、抛关键结论
- **你在记**：一边听一边截图、一边做笔记，注意力被撕裂
- **你在找**：会后回看聊天记录/录屏，想找某一页又找不到

### CUSTRA 的价值（不夸张、可落地）

- **省操作**：从“手动截图 N 次”变成“框选一次 + 结束后去 `output/` 拿成果”
- **更少废图**：只在画面变化时保存（并可定时兜底），更贴近“课/会的节奏”
- **隐私安全**：无需账号，不上传；文件就在你的电脑里

## 5) 产品怎么工作（How it works）⚙️

（页面建议配 3 张图：框选区域 → 运行中 → `output/` 结果）

1. **点击启动**  
2. **在屏幕上框选一个区域**（例如 PPT 区、字幕区、聊天区）  
3. **从首帧开始记录**（首帧必存）  
4. **每秒采样一次**，把当前画面与“上一次保存帧”比较  
5. **触发保存的两条规则**  
   - **变化触发**：变化率 + 变化分布 + 灵敏度  
   - **循环记录（可选）**：每隔 N 秒强制保存一张（循环优先）  
6. **停止后自动归档**：生成一个任务文件夹，里面是 `images/`（可选 `audio/`）

## 6) 功能亮点（Feature Highlights）🌟

- **极简 3 按钮**：启动/音频/设置，上手 10 秒
- **多屏支持但不跨屏**：在哪块屏幕框选，就在那块屏幕记录（不支持跨屏幕选区）
- **时间戳命名**：系统时间（含时区、毫秒）+ 任务内计时，方便对齐回忆与笔记
- **可选系统输出音频（Windows）**：一份 `audio.wav`，便于复听关键段落
- **托盘运行**：最小化到托盘，不占任务栏

## 9) FAQ（落地页常见问题）🙋

### Q：支持MAC和Linux吗？
当前DEMO版本暂不支持，未来可能优化支持。

### Q：这是录屏吗？

不是录屏视频。它会按 1 秒采样，**保存关键变化画面为 PNG**；也可开启循环记录做定时兜底。

### Q：为什么不是直接录屏？

因为很多课/会你只需要“关键页/关键切换”，不需要一个巨大的视频文件；关键帧更易检索、也更适合做笔记。

### Q：文件保存在哪？

本地 `output/<task>/images/*.png`；启用音频则有 `output/<task>/audio/audio.wav`。

### Q：会上传到云端吗？

不会。本地运行，本地输出。
